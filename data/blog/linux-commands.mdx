Go to previous directory: cd -
Chain commands: && (cmd1 && cmd2 && cmd3, etc.)

Search for past uses of command: ctrl + R [...type a command to search for] (press again for 2nd, 3rd, nth result)

-     	cat
-     	split -l
-     	grep
-     	head -n
-     	tail -n
-     	wc -l
  - word count
  - -l number of lines
  - example: wc-l myFile.txt
-     	sed
-     	awk
-     	pbcopy < file.txt
-     	pbpaste > file.txt
-     	scp
